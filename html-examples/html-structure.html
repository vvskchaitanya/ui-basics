HTML Structure and Basic Tags

  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html>: The root element of an HTML page.
  • <head>: Contains meta-information about the HTML document (e.g., the title).
  • <title>: Specifies the title of the web page, which appears in the browser tab.
  • <body>: Contains the content of the HTML document that will be visible on the web page.
  • <h1>: Defines a top-level heading.
  • <p>: Defines a paragraph.
  • <ul>: Defines an unordered list.
  • <li>: Defines a list item.
  • <a>: Defines a hyperlink.
